# 🚀 Portfólio Acadêmico Digital - Flávio Lins

Bem-vindo ao meu portfólio acadêmico! Este projeto foi desenvolvido como parte da minha trajetória no curso de **Análise e Desenvolvimento de Sistemas (ADS)** na **FATEC São José dos Campos**.

O objetivo deste site é centralizar minha formação, competências técnicas e os principais projetos desenvolvidos durante a graduação, servindo como uma ferramenta de apresentação profissional.

## 🛠️ Tecnologias Utilizadas

Para garantir um código limpo, moderno e dinâmico, utilizei:

- **HTML5:** Estrutura semântica das seções.
- **CSS3:** Layout responsivo com Flexbox e Media Queries.
- **JavaScript (Vanilla):** Implementação de lógica dinâmica, incluindo:
  - **Estruturas de Decisão:** Filtros e badges de status para projetos (`if/else`).
  - **Comandos de Repetição:** Renderização automática de cards a partir de arrays de objetos (`for`).
  - **Funções:** Organização e modularização do código.

## 📂 Projetos em Destaque

Neste portfólio, você encontrará detalhes sobre:

1. **Apollo11:** Sistema web para gestão de atestados e avaliação de equipes ágeis (Python/Flask).
2. **Capyjavas:** Ferramenta de gestão de Planos de Desenvolvimento Individual (Java/MySQL).
3. **DM-API:** Plataforma de inteligência de dados para análise de crédito (TypeScript/React) — *Em desenvolvimento*.
